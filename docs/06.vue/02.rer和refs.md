# rer和refs


原来vue2中的ref和$refs是这样的：

```html
<template>
    <div ref="div1"></div>
<template>
<script>
export default {
    mounted() {
        console.log(this.$refs.div1)
    }
}
</script>
```

vue3中的ref和refs是这样的：

```html
<template>
    <div ref="div1"></div>
<template>
<script setup>

import { ref } from 'vue'
const div1 = ref(null)
onMounted(() => {
    console.log(div1.value)
})
</script>
```
